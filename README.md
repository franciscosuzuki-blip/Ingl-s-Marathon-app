# 🏃 English Marathon

Uma plataforma gratuita de aprendizado de inglês para brasileiros através de uma metodologia gamificada.

---

## 🇧🇷 Português (Português Brasileiro)

### Sobre o App

**English Marathon** é uma plataforma web gratuita criada com o objetivo de democratizar o acesso ao aprendizado de inglês para **brasileiros**. Utilizamos uma metodologia inovadora que transforma o aprendizado em uma experiência dinâmica e envolvente, similar a uma corrida emocionante.

### Estrutura de Níveis

Os cursos são organizados em três trilhas, alinhadas com o **CEFR** (Quadro Europeu Comum de Referência):

- 📚 **Inglês Turbo** (2 módulos) - Nível Iniciante
- 📖 **Inglês Exam** (2 módulos) - Nível Intermediário  
- 🚀 **Inglês Race** (2 módulos) - Nível Avançado

### Objetivo

Tornar o aprendizado de inglês **gratuito, acessível e eficaz** para todos os brasileiros, utilizando um método que gamifica a experiência de aprendizado através de uma abordagem baseada em "corridas" de progresso.

### Próximas Atualizações

Os principais recursos e funcionalidades do aplicativo serão apresentados em detalhes na página **[UPDATES.md](./UPDATES.md)**.

## 🇺🇸 English

### About the App

**English Marathon** is a free web platform created to democratize access to English learning for **Brazilians**. We use an innovative methodology that transforms learning into a dynamic and engaging experience, similar to an exciting race.

### Course Structure

Courses are organized in three learning tracks, aligned with the **CEFR** (Common European Framework of Reference):

- 📚 **English Turbo** (2 modules) - Beginner Level
- 📖 **English Exam** (2 modules) - Intermediate Level
- 🚀 **English Race** (2 modules) - Advanced Level

### Objective

Make English learning **free, accessible, and effective** for all Brazilians, using a method that gamifies the learning experience through a "racing progress" approach.
