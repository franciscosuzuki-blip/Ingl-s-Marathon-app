# 🏃 English Marathon App

An innovative language learning platform designed specifically for **Brazilians** to master English through an engaging, gamified racing experience!

---

## 🇧🇷 Português (Brazilian Portuguese)

### Sobre o App

**English Marathon** é um aplicativo revolucionário criado com o objetivo de ajudar **brasileiros** a aprender inglês de forma divertida e interativa. Inspirado na experiência de uma corrida/maratona, o app transforma o aprendizado de idiomas em uma jornada emocionante e motivadora.

### Características Principais

- 🎮 **Gamificação**: Aprenda inglês como se estivesse participando de uma maratona emocionante
- 🇧🇷 **Conteúdo Localizado**: Especialmente desenvolvido para falantes de português brasileiro
- 🏆 **Sistema de Pontos e Desafios**: Ganhe prêmios enquanto avança nos níveis
- 📱 **Interface Intuitiva**: Fácil de usar e acessível para todos os níveis
- 🚀 **Progressão Rápida**: Veja seus resultados melhorando dia após dia

### Objetivo

Democratizar o acesso ao aprendizado de inglês para **brasileiros**, tornando o processo mais divertido, acessível e eficaz através de uma abordagem gamificada.

## 🇺🇸 English

### About the App

**English Marathon** is a revolutionary language learning application specifically designed to help **Brazilians** master English in a fun and interactive way. Inspired by the experience of running a marathon, the app transforms language learning into an exciting and motivating journey.

### Key Features

- 🎮 **Gamification**: Learn English as if you were participating in an exciting marathon
- 🇧🇷 **Localized Content**: Specially developed for Brazilian Portuguese speakers
- 🏆 **Points & Challenges System**: Earn rewards while advancing through levels
- 📱 **Intuitive Interface**: Easy to use and accessible for all proficiency levels
- 🚀 **Rapid Progression**: See your results improve day after day

### Goal

To democratize English learning access for **Brazilians**, making the process more fun, accessible, and effective through a gamified approach.
